COMUNICAÇÃO WEB

Para permitir o monitoramento do sistema em tempo real, foi implementada uma comunicação web utilizando o próprio ESP32 como servidor.

Isso possibilita acessar informações do processo diretamente pelo navegador, sem necessidade de softwares adicionais.

A conexão é feita via Wi-Fi, onde o ESP32 se conecta a uma rede e disponibiliza páginas HTML para visualização dos dados.

Como funciona

📶 Conexão Wi-Fi

O ESP32 se conecta a uma rede sem fio, permitindo acesso remoto ao sistema.

🌐 Servidor Web

Foi criado um servidor interno que responde requisições feitas pelo navegador.

📄 Páginas HTML

O sistema envia páginas como índice, componentes e dados em tempo real.

🔄 Atualização

Os dados são atualizados dinamicamente, permitindo acompanhar o processo ao vivo.

Estrutura da Comunicação

O ESP32 atua como um servidor HTTP, respondendo a diferentes rotas (acessos) feitas pelo navegador.

Cada rota retorna uma página ou informação específica, como: tela inicial, entradas/saídas e dados do sistema.

Também foi criada uma rota dedicada para envio de dados em formato simples, permitindo atualização contínua das informações exibidas.

Por que utilizar comunicação web?

📱 Acesso Fácil

Pode ser acessado por qualquer dispositivo com navegador.

⚡ Tempo Real

Permite acompanhar o funcionamento do sistema instantaneamente.

🔧 Simplicidade

Dispensa softwares externos, utilizando apenas o navegador.

🚀 Profissional

Aproxima o projeto de aplicações reais da indústria.
