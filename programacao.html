PROGRAMAÇÃO

A programação do sistema foi desenvolvida utilizando o ESP32, com o objetivo de automatizar completamente o processo da bancada FESTO.

O sistema realiza a leitura de sensores, identifica o tipo de peça e controla atuadores pneumáticos de forma automática e sequencial.

Além disso, foi implementada uma comunicação via Wi-Fi, permitindo o acesso a um painel web para monitoramento em tempo real.

Lógica do Sistema

🔎 Sensores

Responsáveis por detectar presença, posição e tipo de material da peça.

⚙️ Atuadores

Executam ações físicas como movimentação, avanço e separação.

🔁 Sequência

O processo segue uma ordem definida, garantindo funcionamento correto.

🌐 Web

Permite visualizar dados do sistema em tempo real através do navegador.

🧠 Máquina de Estados

O sistema foi desenvolvido utilizando uma máquina de estados (FSM), dividindo todo o processo em etapas bem definidas.

Cada estado representa uma fase do funcionamento, como: espera da peça, identificação, movimentação e separação.

Essa lógica garante que cada ação aconteça no momento correto, evitando conflitos entre sensores e atuadores.

Com isso, o sistema se torna mais organizado, seguro e fácil de manter, além de facilitar futuras melhorias.

Por que foi feito assim?

📁 Organização

Facilita o entendimento e manutenção do sistema.

🛡️ Segurança

Evita que várias ações ocorram ao mesmo tempo.

🎯 Precisão

Permite saber exatamente em qual etapa o sistema está.

🚀 Evolução

Facilita adicionar melhorias no futuro.
